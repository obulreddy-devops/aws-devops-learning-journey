#-cloud computing ?
Cloud computing means using servers, storage, and database over the internet instead of buying nad maintaning our own server
ex: electricty bill we pay only for what we use.

#-AWS Overview(Amazon web services) is a cloud platform provided by Amazon.
AWS provides:
servers
storage
Database
Networking
Security

#-Regions
A region is a geographical location where AWS data centers are located.
ex:
mumbai
singapore
London

#-Availability Zones
A avaiability zone is a data center inside a Region.
Multiple Availability zones help kep applications running even if one data ceneter fails.
